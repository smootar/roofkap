# RoofKAP - Utah's Premier Roofing Specialists

Professional roofing company website for RoofKAP, serving the Salt Lake City area and surrounding Utah communities.

## Website

Live site: [https://roofkaputah.com](https://roofkaputah.com)

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Hosting**: GitHub Pages
- **Form Handling**: Formspree
- **Domain**: roofkaputah.com

## Features

- Fully responsive design (mobile, tablet, desktop)
- Dark premium theme
- Contact form with spam protection
- Click-to-call phone integration
- Fast loading and optimized images

## Project Structure

```
roofkap/
├── index.html              # Home page
├── about.html              # About page
├── services.html           # Services page
├── contact.html            # Contact page
├── CNAME                   # Custom domain configuration
├── css/                    # Stylesheets
├── js/                     # JavaScript files
└── images/                 # Image assets
```

## Local Development

To run the site locally:

```bash
# Navigate to project directory
cd roofkap

# Start a local server (Python 3)
python3 -m http.server 8000

# Or using PHP
php -S localhost:8000

# Open browser to http://localhost:8000
```
