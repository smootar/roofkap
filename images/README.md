## Image Optimization

Before adding any image to the website:

1. Resize to appropriate dimensions (max 1920px width for hero images)
2. Compress using [TinyPNG](https://tinypng.com/) or [Squoosh](https://squoosh.app/)
3. Target file sizes:
   - Hero images: < 500KB
   - Service images: < 200KB
   - Logo images: < 100KB
